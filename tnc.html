Terima kasih telah mempercayakan momen berharga Anda kepada Dejede Photo & Video. Untuk kenyamanan bersama, mohon perhatikan ketentuan layanan kami setelah pelunasan pembayaran:

1. Status File Digital (All Files / Draft)

Informasi Penting: File yang diserahkan pada H+3 adalah kumpulan file dokumentasi awal (All Files).
  • File yang diterima saat ini masih bersifat Draft dan belum masuk ke tahap proses Detail Editing atau Retouching.
  • Proses penyempurnaan (Detail Editing & Retouch) tetap kami kerjakan lebih lanjut seiring dengan proses produksi album/hasil akhir.

2. Penyerahan File & Media Digital

  • Foto: File dokumentasi akan diserahkan maksimal H+3 setelah pelunasan melalui Google Drive.
  • Video: Hasil video akan di-upload ke YouTube dengan status "Private/Unlisted". Link hanya diberikan kepada klien agar privasi terjaga dan tidak tersebar secara umum.
  • Dalam kondisi jadwal padat, kami tetap berkomitmen mengirimkan file segera sesuai urutan antrean kerja.

3. Batas Penyimpanan & Backup

Peringatan: Link Google Drive bersifat sementara dengan batas waktu penyimpanan maksimal 3 bulan.
  • Setelah 3 bulan, file akan terhapus otomatis untuk manajemen ruang penyimpanan kami.
  • Klien wajib melakukan backup atau copy folder ke Google Drive pribadi agar data aman secara permanen (Langkah cepat bisa lihat tutorial di YouTube).

4. Media Penyimpanan Fisik

  • Standar Paket (Default): File foto dan video akan disertakan dalam media fisik berupa Disk DVD sesuai keterangan paket saat booking.
  • Opsi Flashdisk: Media Flashdisk tersedia dengan adanya biaya tambahan atau sesuai kesepakatan di luar paket standar.
  • Penyerahan media fisik biasanya dilakukan bersamaan dengan penyerahan album fisik.

5. Proses Produksi Album & Cetak

Kualitas maksimal membutuhkan waktu proses yang teliti:

  • Proses pengerjaan album (Editing, Retouching, Layouting, hingga Produksi Cetak) memerlukan waktu minimal 1 bulan dengan batas waktu 1 hingga 2 bulan.
  • Kami menangani banyak klien secara bergantian, sehingga setiap proyek dikerjakan dengan perhatian detail dan etos kerja profesional. Kami harap klien dapat memahami waktu dan proses kreatif yang kami jalankan demi hasil terbaik.
